HTML structure

<!doctype html>
<html></html> - Tags (like a container)
hover an element and it will show what the tag is

<head> - all the metadata of the website (invisible)
<body> - all the visible part for the website

- save it as homepage.html and open it in browser
- we are browsing a current page
