HTTP Status Codes
 - these are informational to know what happens
 - 400 - the client made an error (bad request)
 - 401 - the client is not authorized
 - 403 - the client does not have access to this resource
 - 404 - cannot find the resource find for
 - 500 - something went wrong on the server side
 - 300 - redirecting the request, eg. for domain redirect
